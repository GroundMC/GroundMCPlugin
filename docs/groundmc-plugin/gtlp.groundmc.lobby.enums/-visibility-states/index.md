# VisibilityStates

`enum class VisibilityStates`

### Enum Values

| [NONE](-n-o-n-e.html) | Hides all players. |
| [FRIENDS](-f-r-i-e-n-d-s.html) | Hides all players except for friends. |
| [ALL](-a-l-l.html) | Hides no players == All players are visible. |
